# Mnemos Version

System Name: Mnemos

Version: `0.1.0-first-cultivation`

Status: First usable cultivation template

## Purpose

This file records the version of the Mnemos project-memory system used by this repository.

Mnemos is a collaborative project memory system. The repository is the durable memory. The AI helps organize, maintain, and build upon that memory.

## Compatibility

Designed for:

- GitHub repositories;
- Markdown files;
- AI-assisted project continuity;
- human-readable documentation;
- private, shared, or public project repositories;
- users who may not know how GitHub works.

## Core Features

- `MNEMOS_CONTRACT.md` shared human-AI working agreement;
- `IDENTITY.md` project bootloader;
- `CURRENT_STATE.md` active handover;
- `PROJECT_MAP.md` repository navigation;
- recursive `INDEX.md` files;
- project constitution;
- project journal;
- bookkeeping protocol;
- getting started guide;
- user setup guide;
- cultivation/initialization guide;
- reusable prompts.

## Core Principle

The Mnemos core should remain small.

Everything beyond the core is cultivation.

Cultivation means the AI and user adapt the repository to the actual project through dialogue, use, and observation.

## Upgrade Rule

Do not upgrade a project to a new Mnemos version automatically.

Record major template changes in the project journal first. Promote them to the constitution or version file only after they prove useful.
